征服美洲常见问题及解决方法

声明：因征服美洲是一款老游戏（最早发行于2003年），BUG很多，并且官方已经放弃这款游戏，不再进行更新，2009年官方出版的数字重制版征服美洲依然存在一些问题。本页列举出游戏运行方面常见的一些问题、BUG的解决方法。以下的解决办法，是老玩家经验所得，本站不能保证以下的方法都完全有效，如果依然无法解决玩家的问题，请加入QQ群 602408074 回报您的问题。


　
游戏花屏问题
　


问题描述：在游戏菜单界面和游戏过程中，整个画面呈现绿、紫、红、黑等色块，无法正常显示游戏画面。该问题一般会在 Windows XP 或 Windows 7 操作系统运行旧版（2003年版本）游戏时出现。



本站的重置版征服美洲游戏可以解决所有花屏、黑屏问题。

若要玩旧版本，请根据不同的操作系统，选择不同的补丁或工具。



Windows 8 / Windows 10

在Windows 8 和Windows 10操作系统下，旧版征服美洲的引擎无法运行，打开后会显示黑屏，无法进行游戏；只有重制版征服美洲可以正常进行游戏。

因此，该操作系统下，请下载重制版征服美洲进行游戏。




Windows 7

在Windows 7操作系统下，旧版征服美洲游戏会与一些新版显卡产生不兼容现象，因而产生花屏。如果在游戏进行过程中，关闭Win 7 系统中的explorer.exe（即Windows文档管理器）进程，则游戏不会花屏。

该操作系统下，请先打开“征服美洲花屏解决工具”再进行游戏，在游戏启动的时候，“征服美洲花屏解决工具”会自动关闭explorer.exe进程，再退出游戏后，该工具会自动打开explorer.exe进程。你也可以通过工具中的按钮自己关闭/打开explorer.exe进程。


下载地址：下載地址1 　下載地址2　下載地址3

注意：

　　1、explorer.exe进程关闭后，文件夹、导航栏、桌面图标等都会相应看不到（只能看到一张桌面壁纸），这是正常现场，请勿惊慌；

　　2、若手动将explorer.exe进程关闭后，请勿退出“征服美洲花屏解决工具”，否则explorer.exe进程不容易打开；

　　3、“征服美洲花屏解决工具”开发者即为本网站站长，工具绿色无毒，请放心使用。

Windows 7 系统下还能使用“Win7花屏通用补丁”来解决花屏。

使用方法：将下载后文件夹里的若干个文件复制到游戏根目录下，双击“win7Fix.exe”文件启动后，选择征服美洲启动文件（dmcr.exe），然后选择运行游戏即可开始游戏。

游戏过程中突然跳出并弹出提交BUG的网页
　


问题描述：在游戏进程过程中，会突然弹出到桌面，游戏程序奔溃，浏览器打开一个提交BUG的本地网页（如下图）。



该BUG是征服美洲游戏引擎的BUG，目前无法解决，建议可以将游戏分辨率调成显示器支持的最大分辨率或最大分辨率的下一级分辨率。


　
启动游戏时弹出对话框，内容为：
Dilplay mode 1024×768×8 not found. Cossacks should not run.
　


问题描述：启动游戏时弹出对话框，显示内容为“Dilplay mode 1024×768×8 not found. Cossacks should not run.”（如下图），若点击重试，依然可以进入游戏，但进入游戏后无法选择分辨率。



打开游戏文件夹中 dmcr.exe 文件的属性，在“兼容性”分页中，勾选“简化的颜色模式”这一选项，并在下面的下拉菜单选项中选择默认的“8位(256)色”一项。如下图。


　
无法正常启动，并弹出对话框内容为：
应用程序无法正常启动(0xc0000022)。请单击“确定”关闭应用程序。
　


问题描述：启动游戏时弹出对话框，显示内容为“应用程序无法正常启动(0xc0000022)。请单击“确定”关闭应用程序。”（如下图），无法进入游戏。



打开Windows 的“控制面板”——“程序”——“启用或关闭Windows功能”，此时会弹出Windows功能对话框，拉到最下面，点击"+”展开“旧版组件”，之后勾上“DirectPlay”，再点击确定；随后会完成组件功能的修改，直到出现“Windows 已完成请求更改”的对话框出现，关闭该对话框，启动游戏即可正常运行。


　
在游戏界面，运行卡顿，或单位卡住不动，不能进行任何操作。
　


问题描述：进入游戏后，在运行战役、单人游戏、多人游戏、编辑器等游戏操作界面，游戏过程卡顿、或卡死不动。



打开游戏文件夹中 dmcr.exe 文件的属性，在“兼容性”分页中，勾选“以兼容模式运行”这一选项，在下拉菜单中选择“Windows XP SP3”。


　
进入游戏后，只有黑屏界面，同时有背景声音，点击也无反应
　


问题描述：进入游戏后，只有黑屏界面，同时有背景声音，点击顺便无反应。



打开游戏文件夹中 dmcr.exe 文件的属性，在“兼容性”分页中，勾选“以兼容模式运行”这一选项，在下拉菜单中选择“Windows XP SP3”。

或者：打开征服美洲游戏目录，将游戏跟目录下的“Video”删除或改成其他名字。


　
打开游戏后，界面显示不全
　


问题描述：打开游戏之后，菜单界面以及游戏界面显示不全，只显示左上方一部分。



打开游戏文件夹中 dmcr.exe 文件的属性，在“兼容性”分页中，勾选“在 640 x 480 屏幕分辨率下运行”与“覆盖高 DPI 缩放行为”两个选项，并在下面的下拉菜单选项中选择默认的“应用程序”一项。如下图。


　
打开游戏设置，无法调整分辨率
　


问题描述：打开游戏的设置中，调整分辨率的选项为空，无法修改分辨率。



打开游戏文件夹中 dmcr.exe 文件的属性，在“兼容性”分页中，勾选“在 640 x 480 屏幕分辨率下运行”。


　
修改游戏分辨率、游戏速度时，游戏会奔溃
　


问题描述：修改游戏分辨率、游戏速度时，游戏会立刻奔溃，跳出到桌面。



打开游戏文件夹中 dmcr.exe 文件的属性，在“兼容性”分页中，勾选“以兼容模式运行”这一选项，在下拉菜单中选择“Windows XP SP3”。
