MOD介绍

　　MOD即游戏模组，英文多简称为“MOD”（全称“Modification”），“修改”的名词含义，常为第三方（以玩家为主）进行修改并附加到原游戏上，往往修改了一些属性，增加国家、单位、建筑或增加了战役、任务以及画面修改等。
　　曾有不少国外玩家进行过修改，制作了一些MOD。本站收录了若干个MOD和补丁。其中有些MOD很完善，可以作为单独的游戏，有些MOD仅仅修改了一些单位属性。

Introduction of MOD

　　Some players may modify the game and make some patch files. It often changed some of the properties, increasing the country, unit, building, or battle, tasks, and picture modification.
　　There have been many players modified, they made some MODs. This site contains a number of MODs. Some of these MOD is very perfect, can be used as a separate game, some MOD is a semi-finished product, but you can experience, and some MOD only modified a few properties, can be regarded as a patch class.
